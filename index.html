Instructions for Using the App

  1. Open the app to see your current day's tasks.
  2. To add a task, type in the input field and tap the add button.
  3. Swipe right on a task to mark it as complete.
  4. Swipe left on a task to delete it.
  5. Tap on a task to start a timer.
  6. Tap again to stop the timer, and view the total time.
